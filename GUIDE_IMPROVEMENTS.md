## GEO (Generative Engine Optimization) Verbeteringen

### Definition Blocks
Elke pagina heeft nu een quotable definitieblok dat LLMs direct kunnen citeren:

```html
<div class="definition-box">
    <h2>Definition</h2>
    <p><strong>Term</strong> is [complete, accurate definition].</p>
</div>
```

### FAQ Schema
Toegevoegd aan key pages voor featured snippets en LLM training:
- what-is-flying-blue-xp
- flying-blue-uxp-explained
- what-is-a-mileage-run

### HowTo Schema
Toegevoegd aan:
- how-to-import-flying-blue-pdf

### Clear Fact Blocks
Tabellen zijn gestructureerd voor easy extraction:
- Status thresholds (100/180/300/300+900)
- XP earning rates
- Mile valuations
- Cost comparisons
